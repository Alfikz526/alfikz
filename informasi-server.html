Informasi Server Alfikz SMP

Aturan Server

1. Tidak melakukan kecurangan (cheating)

Pastikan untuk bermain dengan fair dan tidak menggunakan cheat atau bug untuk keuntungan pribadi.

2. Hormati pemain lain dan staff

Jaga sikap sopan santun dalam berinteraksi dengan pemain lain. Jangan melakukan tindakan yang merugikan pemain lain.

3. Tidak menggunakan bahasa kasar

Hindari penggunaan bahasa yang tidak pantas atau menyakiti perasaan orang lain di server.

4. Tidak merusak bangunan orang lain

Hargai bangunan orang lain dan jangan merusaknya tanpa izin.

5. Bangun bangunan yang baik

Bangunlah bangunan yang baik dan tidak merusak pemandangan server.

6. Tidak melakukan spam

Hindari melakukan spam di chat server.

7. Tidak melakukan iklan server lain

Hindari melakukan iklan server lain di dalam server kami.

8. Tidak melakukan tindakan yang merugikan server

Jangan melakukan tindakan yang merugikan server kami, seperti DDOS atau tindakan merusak lainnya.

9. Ikuti aturan tambahan dari staff

Admin berhak menambah atau mengubah aturan sewaktu-waktu demi menjaga kenyamanan saat bermain.

Fitur Server

Server Alfikz SMP memiliki berbagai fitur seru yang akan membuat pengalaman bermain Minecraft semakin menyenangkan. Beberapa fitur utama server kami antara lain:

  • Mode Survival: Nikmati petualangan di dunia Minecraft dengan mode survival yang menantang.
  • Mini Games: Bergabunglah dengan teman-temanmu untuk bermain berbagai mini games seru.
  • Sistem Ekonomi: Bangun kerajaanmu dengan sistem ekonomi yang memungkinkan kamu untuk membeli dan menjual barang.
  • Pembelian Rank: Dapatkan akses lebih banyak fitur dengan membeli rank di server kami.
  • Event Berkala: Ikuti event-event menarik yang diadakan secara berkala untuk mendapatkan hadiah menarik.
  • Komunitas Aktif: Bergabunglah dengan komunitas kami yang aktif dan ramah untuk berbagi pengalaman dan tips bermain.
  • Support 24/7: Tim support kami siap membantu kamu kapan saja jika mengalami masalah.
  • Server Stabil: Server kami memiliki uptime yang tinggi dan performa yang stabil untuk pengalaman bermain yang lancar.
  • Backup Data: Data server kami selalu di-backup secara berkala untuk menjaga keamanan data kamu.
  • Update Berkala: Kami selalu melakukan update untuk menambah fitur baru dan memperbaiki bug.
  • Custom Plugin: Server kami menggunakan plugin custom yang membuat pengalaman bermain semakin unik dan menarik.
  • Map yang Menarik: Nikmati dunia Minecraft yang telah dimodifikasi dengan map yang menarik dan penuh tantangan.

Cara Bergabung dengan Server

Untuk bergabung dengan server Alfikz SMP, kamu hanya perlu mengikuti langkah-langkah berikut:

  1. Download dan install Minecraft di perangkat kamu.
  2. Pilih "Multiplayer" dan klik "Add Server".
  3. Masukkan IP server: play.alfikzSMP.id.
  4. Hubungkan dan mulai bermain!
